**Aplicativo de Grade de Cursos**
----------------------------

Um aplicativo Android desenvolvido em Kotlin com Jetpack Compose, apresentando uma grade (grid) de temas de cursos.
Cada item exibe uma imagem, o título do tema e a quantidade de cursos relacionados.


**Sobre o Projeto**
-------------------

Este projeto foi criado para praticar o uso de layouts em grade (LazyVerticalGrid) com o Jetpack Compose, explorando o uso de componentes reutilizáveis, imagens vetoriais, e dados estáticos em uma interface organizada e moderna.
O foco está em compreender como exibir coleções de itens de forma responsiva e eficiente na tela.


**Pré-requisitos**
------------------

Conhecimento básico em Kotlin.

Noções sobre Jetpack Compose e seus componentes.

Entendimento de LazyVerticalGrid e Card no Compose.

Saber executar aplicativos no Android Studio (emulador ou dispositivo físico).

**Como Executar**
-----------------

Instale o Android Studio (versão mais recente recomendada).

Clone ou baixe este repositório.

Abra o projeto no Android Studio.

Compile e execute o aplicativo em um emulador ou dispositivo físico.

**Funcionalidades Principais**
------------------------------

Exibição de uma grade de cursos com duas colunas.

Cada item contém:

Uma imagem ilustrativa.

O nome do tema do curso.

O número de cursos associados.

Interface totalmente construída com Jetpack Compose.

Layout adaptável e responsivo.

**Tecnologias Utilizadas**
--------------------------

Kotlin

Jetpack Compose

Material Design 3

Android Studio
